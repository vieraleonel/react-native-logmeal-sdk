# react-native-logmeal-sdk

Logmeal's SDK integration for react-native

## Installation

```sh
npm install react-native-logmeal-sdk
```

## Usage

```js
import { LogmealSdkView } from "react-native-logmeal-sdk";

// ...

<LogmealSdkView color="tomato" />
```
